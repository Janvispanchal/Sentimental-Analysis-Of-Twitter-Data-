# Twitter Sentiment Analysis 📊

## 📌 Overview

This project analyzes tweets related to COVID-19 vaccination and performs sentiment analysis using basic NLP techniques. It classifies tweets into Positive, Negative, or Neutral sentiments.

---

## 🛠️ Tools & Libraries

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* NLTK, TextBlob
* Scikit-learn
* WordCloud

---

## 🔍 Steps Performed

1. Data loading and exploration
2. Text cleaning and preprocessing

   * Removing special characters
   * Tokenization
   * Stopword removal
   * Stemming
3. Sentiment analysis using TextBlob
4. Visualization using WordCloud and charts
5. Model building using Logistic Regression
6. Model evaluation

---

## 📂 Dataset

* vaccination_tweets.csv

---

## ▶️ How to Run

1. Create virtual environment:
   python -m venv venv

2. Activate:
   venv\Scripts\activate

3. Install dependencies:
   pip install -r requirements.txt

4. Run notebook:
   main.ipynb

---

## 📈 Result

The model predicts sentiment of tweets and helps understand public opinion on vaccination.
